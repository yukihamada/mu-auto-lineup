1. 项目背景与本询价的定位Background & Scope of This RFQ

MO-1 是一款以纯电动平台为基础的移动办公室（"会移动的总部"）。其核心价值不在于价格，而在于停车状态下的办公生产力：以接近固定办公室的环境进行办公作业。

MO-1 is a mobile office ("a head office on wheels") built on a battery-electric platform. Its differentiation is not price but productivity while parked.

关键特征：通信采用卫星 + 5G/4G + Wi-Fi 三重冗余；供电采用驱动电池 + 办公副电池 + 太阳能，设计目标为无补给 3 天（72 小时）；停车专用超低功耗 HVAC。目标在日本以特种用途"8 号牌（事务用途车）"完成型式与登记（待确认）。全车为右舵（RHD）。

本询价定位 / Positioning 本函为技术可行性评估 + 预算性报价（feasibility + budgetary）。所列数值均为设计目标值（design target / requirement）·非保证值。我方理解多项条件（续航、电力收支、认证适用性、8 号牌要件、图纸）尚未冻结，不要求贵司在未确定信息上给出确定价格；请以"可行性判断 + 概算（budgetary）+ 前提 / 除外"形式回复。确定报价在 Phase 0 图纸与条件冻结后另行进行。 This is a feasibility + budgetary RFQ. All figures are design targets, not guaranteed. We do not expect firm prices on unfrozen items; please reply with feasibility + budgetary estimates + assumptions/exclusions. Firm quotation follows Phase 0.
